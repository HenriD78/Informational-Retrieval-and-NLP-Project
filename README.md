# NLP Project 2

This project is a web application that uses natural language processing to analyze and process text data.

link to the [video](https://youtu.be/6hUcVQAINdU) presenting the project :

## Installation

1. Clone the repository:
```bash
git clone <repository-url>
```

2. Create a virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
streamlit run streamlit_app.py
```

## Project Structure

- `streamlit_app.py`: Main application file
- `pages/`: Additional pages for the application
- `data/`: Data files
- `models/`: Model files
- `logs/`: Log files
